temperature = 0 — logits pass through unchanged

The code only divides logits by temperature when it is > 0. At 0, logits are used as-is. No randomness is introduced — but the filters below still reshape the candidate set.

No effect on ranking. Global argmax is still sophisticated (40%). Temperature does not participate in pool building here; it would only matter for stochastic sampling.
